## 导入器
提供方便你将`Chrome`书签导入为本程序使用的工具。

### 使用方式
```bash
go run main.go [参数]
```

```txt
usage: main --filename=FILENAME [<flags>]

Flags:
      --help               Show context-sensitive help (also try --help-long and --help-man).
  -f, --filename=FILENAME  filename of Chrome's bookmarks
  -o, --output=OUTPUT      output filename for flare bookmarks, print to stdout if not set
      --smooth             smooth the output
      --version            Show application version.

```
